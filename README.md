# GeradorInsert
Gerador utilizado para fins acadêmicos afim de ajudar alunos em suas disciplinas de banco de dados, com ele é possível preencher campos dos mais diversos tipos consultando o manual que está disponivel no cabeçalho da página(Ajuda).
O Gerador não é nenhum sistema profissional capaz de resolver os diversos problemas das máterias de Banco de Dados e sim um 'ajudante' que lhe pode adianta algum tempo gerando inserts aleátorios que gastariam um bom tempo digitando na mão mesmo com o 'Ctrl+C'.
